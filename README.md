# allenjohn.me

Source code for my personal portfolio and blog.

## Stack
- **Astro**
- **Vanilla CSS**
- **Content Collections**
- **GitHub Pages**

## Dev
```bash
npm install
npm run dev
```

## Blog
To write a post, I add a `.md` file to `src/content/blog/`:

```markdown
---
title: "Post Title"
description: ""
date: 2026-03-24
tags: ["tag"]
draft: true
---
```
