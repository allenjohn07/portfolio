# Hello World

This is a sample blog post to test the setup. You can delete this file and start writing your own posts!

Each blog post is a simple `.md` file in `src/content/blog/`. Just add YAML frontmatter at the top:

```yaml
---
title: "Your Post Title"
description: "A short description"
date: 2026-03-24
tags: ["tag1", "tag2"]
draft: false
---
```

Then write your content in Markdown below. Code blocks get syntax highlighting automatically.
